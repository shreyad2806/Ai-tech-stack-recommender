🚀 StackMind AI

AI-assisted system design and architecture visualization platform.

StackMind helps users transform startup ideas into:

- Tech stack recommendations
- Architecture diagrams
- AI workflow visualizations
- Deployment suggestions
- Development roadmaps

Built using React, FastAPI, Groq API, and React Flow.

# 🚀 Local Setup

## Clone Repository

```bash
git clone https://github.com/shreyad2806/Ai-tech-stack-recommender.git

cd Ai-tech-stack-recommender
```

## Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn main:app --reload
```

Create `.env`

```env
GROQ_API_KEY=your_api_key
```

## Frontend

```bash
cd frontend

npm install

npm run dev
```
